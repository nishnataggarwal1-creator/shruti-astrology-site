# Shruti Aggarwal Astrology

Deploy-ready Vite + React website.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy on Vercel
1. Upload this folder to a GitHub repo.
2. Import the repo into Vercel.
3. Click Deploy.

## Edit later
Open `src/App.jsx` and update:
- BUSINESS details
- SERVICES list
- Reviews
- FAQ
